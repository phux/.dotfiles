# Knowledge Base

This directory contains codified lessons learned from agent sessions. It acts as the institutional memory for the project.

## Entry Convention
Every new entry in these files MUST be prefixed with the date in `[YYYY-MM-DD]` format. This allows future agents to identify outdated or superseded knowledge.

Example:
### [2026-04-03]
- **Naming Convention**: Models should use PascalCase for class names and snake_case for methods.
- **Quirk**: The `/health` endpoint requires a trailing slash or it returns 301.
