*   `playbook.yml`: Main Ansible playbook.
*   `roles/`: Individual Ansible roles for specific tools/aspects.
*   `group_vars/`: Variables for host groups (`group_vars/all/vars.yml` for global variables).
*   `hosts`: Ansible inventory file.
*   `Makefile`: Shortcuts for common tasks.
*   `.gemini/`: Gemini CLI related files.
*   `.git/`: Git repository.
*   `scripts/`: Contains `update-versions.sh`.
